# Definition of Done

## Criterios
- Entregable implementado segun alcance.
- Validaciones ejecutadas sin errores.
- Trazabilidad actualizada (status, log, changelog, RTM).
- Riesgos residuales documentados.

## Regla de cierre
El estado final operativo previo a revision humana es `READY_FOR_VALIDATION`.
