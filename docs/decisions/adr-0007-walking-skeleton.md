# ADR-0007 — Walking skeleton temprano

## Contexto
Evitar construir por capas sin integración.

## Decisión
Los planes maestros incluirán temprano un “walking skeleton”:
- Auth + selección branch
- 1 caso core (crear cita) con DB real
- Runbook + smoke

## Consecuencias
- Se entregan incrementos integrados desde el inicio.

2026-02-10
