# ADR-0006 - Principios UX

- Fecha: 2026-02-10
- Estado: vigente

## Contexto
La operacion clinica requiere velocidad y baja carga cognitiva.

## Decision
Priorizar claridad, eficiencia y feedback accionable en flujos core.

## Consecuencias
- Mejor adopcion por usuarios operativos.
- Requiere pruebas tempranas de usabilidad.
