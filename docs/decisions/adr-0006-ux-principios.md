# ADR-0006 — Principios UX vendible

## Contexto
Demo debe ser convincente en pocos minutos.

## Decisión
- UI en español
- Flujo core accesible en 2 clics
- Errores humanos + validaciones claras
- Acciones sensibles siempre con modal “motivo”

## Consecuencias
- Se prioriza UX sobre features exóticos.
