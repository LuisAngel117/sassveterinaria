# ADR-0006 — UX Principios

## Contexto
- Demo portafolio: debe ser usable y “vendible” en local.

## Decisión
- UI en Español.
- Flujo core visible desde temprano: “crear cita → atender → cerrar → facturar”.
- Errores claros y consistentes.
- Seeds demo para probar en 2–3 minutos.
