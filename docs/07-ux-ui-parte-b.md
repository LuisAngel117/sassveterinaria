# 07 - UX/UI Parte B

## Lineamientos visuales iniciales
- Jerarquia tipografica consistente.
- Estados claros: vacio, carga, error y exito.
- Contraste legible para uso continuo en clinica.

## Accesibilidad
- Navegacion por teclado en vistas core.
- Labels explicitos en formularios.
- Mensajes de validacion junto al campo.

## Entregables UX
- Mapa de flujo por modulo.
- Wireframes de agenda y pacientes.
- Definicion de componentes base.
